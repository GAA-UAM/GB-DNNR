# Ninja-GB-DNN

Ninja-Gradient Boosted - Deep Neural Network

Ninja-GB-DNN is the python library for working with Gradient Boosted - Deep Neural Networks for Multi/Single Output Regression problems (Ninja-GB-DNN).
